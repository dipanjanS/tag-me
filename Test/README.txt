TEST DATASET DESCRIPTION:
	Test data consists of
		i)	Folder 'Images' containing Test images
		ii)	Features file 'feature_vectors.txt': consisting of feature vectors for all
			the test images. Each row represents an image (instance) with the first 
			column being image name, remaining 240 columns being the feature-vector
			of the image.
		
******************************************************************
OUTPUT FORMAT:
For final submissions submit text file containing Image name and label/class 
to which the corresponding image belongs to (in a format identical to '/Training/labels.txt'),
must be uploaded in the 'Upload Files' tab on the website. You have to submit the code also, in the format described in instructions.txt
